absolute与float一起使用：float失效。
        .brother{
            position:absolute;
            float:left;    //失效
        }
注意position:absolute;"块状化"

元素一旦设置position属性设置为absolute或是fixed，其display属性值就是block或是tabal。

position:absolute;具有"包裹性"，也就是尺寸包裹，同时具有自适应性。
        .wrap{
            display : inline-block; //没有必要
            position:absolute;
        }
absolute的包含块

普通元素的百分比宽度：相对于父元素的content box宽度计算的。

绝对元素的百分比宽度：相对于第一个position不为static的祖先元素计算的。
